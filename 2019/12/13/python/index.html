circle(radius,extent,个数)
     radius ：弧形半径

当radius值为正数时，圆心在当前位置/小海龟左侧。
当radius值为负数时，圆心在当前位置/小海龟右侧。
extent ：弧形角度。当无该参数或参数为None时，绘制整个圆形
当extent值为正数时，顺小海龟当前方向绘制。
当extent值为负数时，逆小海龟当前方向绘制。
